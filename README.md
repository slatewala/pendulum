# Pendulum

Swing on a rope. Tap to release at the perfect arc. Land on the green platform.

## Play

- The ball swings under simulated gravity
- Tap = let go of the rope
- Land on the platform = +1 point and a fresh swing
- Miss = falling

Platforms get narrower as your score climbs.

## Build

```
flutter pub get
flutter run
flutter build apk --release
```
